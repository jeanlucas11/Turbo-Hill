# 🚗 Turbo Hill

Turbo Hill é um jogo 2D de corrida desenvolvido na Unity, inspirado em jogos como Hill Climb Racing. O objetivo do jogador é percorrer a maior distância possível enfrentando terrenos acidentados, administrando o combustível do veículo e realizando melhorias para alcançar novas áreas.

---

## 🎮 Sobre o jogo

No Turbo Hill, o jogador controla um veículo que deve atravessar diferentes tipos de terreno enquanto coleta moedas e combustível pelo caminho. As moedas obtidas podem ser utilizadas para comprar melhorias, tornando o veículo mais rápido, estável e eficiente.

O desafio consiste em equilibrar velocidade, consumo de combustível e controle do veículo para alcançar distâncias cada vez maiores.

## ✨ Funcionalidades

- Consumo de combustível durante a corrida.
- Coleta de moedas.

- Interface com informações do jogador.
- Sistema de distância percorrida.
- Loja para melhorias do veículo.

---

## 🔧 Sistema de Upgrades

O jogador poderá melhorar diversos componentes do veículo:

- 🚀 **Motor** — aumenta a velocidade máxima.
- ⛽ **Tanque** — aumenta a capacidade ou reduz o consumo de combustível.
- 🛞 **Pneus** — melhora a aderência ao terreno.
- ⚙️ **Suspensão** — aumenta a estabilidade do veículo.
- 💨 **Nitro** *(planejado)* — fornece aceleração temporária.
